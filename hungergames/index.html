Minecraft Classical Hunger Games Server

MC-HG is a project to bring the classic Hunger Games server gamemode back to Minecraft Multiplayer (not the Survival Games that arose after it and that we have today). You can join the Discord Server here: https://discord.com/bFB3ge9GmE

I have singlehandedly developed this plugin by myself, though outdated by some versions (it is currently only barebones with its fully functional features).
